# Full Stack Developer Portfolio

This repository contains a static portfolio website built with a single HTML file and styled using Tailwind CSS. The page showcases an aspiring full stack developer profile, skills, featured projects, and contact links.

## Project Overview

- **Main file:** `profolio.html`
- **Type:** Static portfolio website
- **Design:** Modern, dark themed, responsive layout
- **Technologies:** HTML, Tailwind CSS, Font Awesome, EmailJS, Google Fonts

## Features

- Hero section with developer introduction and CTA buttons
- Responsive navigation with smooth scrolling
- About section with personal profile details and status
- Skills section grouped into frontend, backend, and data/devops categories
- Featured projects section with descriptions, technology tags, demo links, and code links
- Contact section with email CTA and external profile links
- Custom Tailwind configuration with brand colors and typography
- Visual effects: blurred gradient backgrounds, card hover animation, glassmorphism styling

## Tech Stack

- HTML
- Tailwind CSS (via CDN)
- Font Awesome icons
- Google Fonts (Inter)
- EmailJS browser script for contact support

## Usage

1. Open `profolio.html` in a web browser.
2. The page should render directly without a build process.
3. Update the content, images, and links as needed.

## Customization

- Replace the placeholder email address in the `mailto:` link for the contact button.
- Update the profile image references such as `profile.jpeg`.
- Replace project image files and links to match your own work.
- Modify the Tailwind color palette in the embedded `tailwind.config` section.

## Notes

- The page is designed to work as a standalone static website.
- If you want it to be served from a web server, simply host the HTML file and related assets.
- The project does not include a build system or package manager configuration.
